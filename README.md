# Usage

In the following commands, if you don't have `pnpm`, you can use `npm` instead.

1. `docker compose up -d`
2. `pnpm i`
3. `pnpm run dev`
4. Open `http://localhost:3000` in your browser
5. Log in as one of othe three built-in users

| user | password  |
|------|-----------|
| a@a  | testing_a |
| b@b  | testing_b |
| c_c  | testing_c |

> [!TIP]
> You can log in as multiple users at once if you open `http://localhost:3000` in a new "Private Browsing" window,
> or in a new browser
